# 커리어 던전 Frontend

이력서를 기반으로 AI 면접관과 실전형 기술 면접을 연습하는 게임형 서비스 **커리어 던전**의 프론트엔드 저장소입니다.

## 주요 화면

- 회원가입 및 로그인
- 이력서 업로드
- 면접관 목록과 잠금·해금 상태
- 질문, 답변, 꼬리질문으로 이어지는 대화형 면접
- 최종 충족률 게이지와 면접 결과

## 기술 스택

- React
- TypeScript
- Tailwind CSS
- REST API

## 사용자 흐름

1. 로그인 후 이력서를 업로드합니다.
2. AI 면접관을 선택합니다.
3. 이력서 기반 질문에 답변합니다.
4. 필요한 경우 꼬리질문에 추가로 답변합니다.
5. 최종 충족률과 면접 결과를 확인합니다.

## 개발 상태

현재 MVP 개발 준비 단계입니다. 세부 실행 방법과 환경 변수는 프로젝트 구현에 맞춰 추가할 예정입니다.
