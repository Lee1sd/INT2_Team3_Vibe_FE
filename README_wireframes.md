# Career Dungeon - UI & Flow Diagrams

The `wireframes/` directory contains the generated screen flowcharts and wireframe layouts for the Career Dungeon application.

## Files
1. **detailed_flowchart.svg**: The complete screen flow logic, matching the requested scenario:
   - Login -> Main Lobby -> My Page (Upload) -> Main Lobby -> Interview -> Result (with Badge branching).
2. **flowchart.svg**: A simpler high-level node map.
3. **wireframes_1_to_3.svg**: Layout mockups for Login, Resume Upload, and Dungeon Map.
4. **wireframes_4_to_6.svg**: Layout mockups for My Page, Interview Process, and Result Report.
5. **wireframes.html**: A web page to easily view all the SVG mockups and flows in a browser.

You can directly view the SVG files in any modern web browser or embed them into your IT planning document.
